# Home Affordability Estimator

A full-stack web application that estimates the maximum home price a buyer can afford based on their desired monthly payment, down payment, loan terms, and additional housing costs.

## Features

* Calculates maximum affordable home price
* Supports property taxes, homeowners insurance, PMI, and HOA fees
* Frontend input validation
* Backend request validation
* Unit tested with JUnit

## Tech Stack

### Frontend

* React
* TypeScript
* Vite

### Backend

* Java
* Spring Boot
* Maven
* JUnit
